<ul style="columns: 1;-webkit-columns: 1;-moz-columns: 1;list-style: square outside none;">

       +4871 3202109 instead. </li>
  <li> Please use the current version of the copyright available <a href="ftp://ftp.springer.de/pub/tex/latex/llncs/LNCS-Springer_Copyright_Form.pdf" target="_blank">here</a> and enter <i>ICITS 2015 - 8th International Conference on Information-Theoretic Security</i> as conference name and <i>Stefan Wolf</i> as volume editor.</li>
</ul> 
 
<h4>Camera Ready Paper</h4>

<ul style="columns: 1;-webkit-columns: 1;-moz-columns: 1;list-style: square outside none;list-style: square outside none;">
  <li> The files have to be uploaded using your EasyChair author account. </li>
  <li> Generally the page limit is <b>18 pages</b>. 
  <ul style="columns: 1;-webkit-columns: 1;-moz-columns: 1;list-style: circle outside none;">
    <li> If you cannot fit into this limit please contact <a href="mailto:loc@icits2015.net" target="_blank">loc@icits2015.net</a>. </li>
    <li>  Please do not adjust  the font size or the page layout in order to reduce the number of pages 
     -- they will be converted to the standard values anyway. </li>
  </ul></li>
  <li> Please follow strictly the <a href="http://www.springer.de/comp/lncs/authors.html" target="_blank">author instructions</a> of Springer-Verlag
    when preparing the final version. In particular you are asked to use the <a href="ftp://ftp.springer.de/pub/tex/latex/llncs/latex2e/llncs2e.zip" target="_blank">LNCS styles</a>. </li>
  <li> Whenever possible, use only standard packages. All <b>nonstandard packages must be included</b> in the zipfile.</li>
  <li> The uploaded files consist of 
    <ol style="columns: 1;-webkit-columns: 1;-moz-columns: 1;">
      <li>either a zipped file containing all your LaTeX sources</li>
      <li>PDF version of your camera-ready paper.</li>
    </ol>
  </li>
  <li> The publisher has introduced an extra control loop: once
    data processing is finished, they will contact all corresponding
    authors and ask them to check their papers.</li>
</ul>
